# Лабораторные работы

### Общая цель
В ходе лабораторных работ необходимо найти временной ряд и построить его качественный прогноз.

### Цели лабораторных работ
1. Определение типа процесса - определить к какому типу процесса относится временной ряд для возможности построения в дальнейшем качественного прогноза.
2. Структурные изменения - проанализировать структуру временного ряда на наличие структурных изменений для того, чтобы корректно смоделировать, а затем спрогнозировать динамику процесса.
3. Выделение детерминированных компонент из структуры ряда - определить наличие или отсутствие тренда и сезонной компоненты в структуре временного ряда и при наличии выделить их корректным способом.
4. Смоделировать показатель с помощью ARIMA-инструментов для построения в дальнейшем качественного прогноза.
5. Модели условной гетероскедастичности - построение моделей условной гетероскедастичности "ARCH" / "GARCH".
6. Построение прогноза - прогнозирование исследуемого ряда на основе построения моделей "ARIMA" и моделей условной гетероскедастичности "ARCH" / "GARCH".
